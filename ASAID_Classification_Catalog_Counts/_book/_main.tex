\hypertarget{mobile-species-classification-catalog}{%
\chapter*{Mobile species classification catalog}\label{mobile-species-classification-catalog}}
\addcontentsline{toc}{chapter}{Mobile species classification catalog}

This Classification Catalog shows example images for all labels used for the point-scoring method in the Antarctic Seafloor Annotated Imagery Database (AS-AID). It aims to give users an overview of the labels. This catalog represents a snapshot of the database at the time of publication of the associated data-paper.
The labels used are based on the Australian Morphospecies Catalog, which is an extension of the CATAMI classification scheme. For more information, the CATAMI paper can be found under this \href{https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0141039}{link}

Up to nine randomly selected images are presented for each classification, alongside a table with the following information:

\begin{itemize}
\tightlist
\item
  AMC\_ID: matching identifier in the Australian Morphospecies Catalog, and extension of the CATAMI classification.\\
\item
  Found\_in\_surveys: a list of all surveys in AS-AID in which this organism has been detected\\
\item
  Total\_annotations: the total number of annotations for this label\\
\item
  VME-taxa: whether this label is identified as a CCAMLR Vulnerable Marine Ecosystem taxa
\end{itemize}

Please direct any enquiries, questions or comments to \href{mailto:Jan.Jansen@utas.edu.au}{\nolinkurl{Jan.Jansen@utas.edu.au}}
